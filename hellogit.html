Khi muốn tìm hiểu code của trang web thì cách lấy toàn bộ code của web là một trong những việc cần thiết để ta tìm hiểu về trang web. Hôm nay cachthietkeweb.vn sẽ viết bài hướng dẫn về cách lấy toàn bộ code của web mới nhất 2020

Cách lấy html của 1 web

Để lấy được html của một web site, bạn sẽ cần thực hiện qua 4 bước chính:
Bước 1: Chọn website bạn cần lấy code, sau đó mở xem nguồn trang.

Bước 2: Chọn lấy mã code

Bước 3: Tạo thư mục để lưu trữ (Dạng Notepad ++)

Bước 4: Dán nội dung bạn đã chọn vào thư mục. Hoàn tất việc lấy code của web.

Dưới đây mình sẽ miêu tả cụ thể từng bước để các bạn dễ hình dung và thao tác nhé.

Đầu tiên bạn cần tìm cho mình một trang website mà bạn muốn copy. Chẳng hạn trong bài tut này chúng tôi sẽ get web cachthietkeweb.vn để sử dụng ví dụ. Nào chúng ta cùng bắt đầu

Bước 1 : Bạn nhấn chuột phải trên màn ảnh và chọn “Xem nguồn trang” hoặc “View page source” giống như trong hình. Còn có một hướng dẫn khác nữa là bạn hãy bấm tổ hợp phím Ctrl + U.

Ảnh minh họa: phương pháp để thấy mã HTML của web đó

Sau đó bạn sẽ được chuyển đến một tab chứa các mã code html.

Ảnh minh họa: : Trang tab chứa những đoạn code của web

Bước 2: Bạn hãy bôi đen những mã code html, có thể thực hiện bằng 2 cách:

Cách 1 : Sau khi bạn mở tab chứa mã code html lên thì hãy nhấn tổ hợp phím Ctrl + A để bôi đen toàn bộ.

Ảnh minh họa: Bôi đen code theo cách 1

Cách 2 : Giữ im chuột trái và dẫn xuống để bôi đen mã code HTML (Giữ chuột đến hết nội dung)

Ảnh minh họa: Bôi đen mã code theo cách 2

Bước 3 : Để lưu mã code html vừa sao chép. Đầu tiên bạn cần tạo một thư mục để lưu. Sau đó ấn vào thư mục, nhấn chuột phải, chọn Text Document (như hình).

Ảnh minh họa : Tạo kênh lưu trữ HTML bằng Text Document

Sau khi lưu thì hãy đổi tên mục Text Document thành Index.html

Ảnh minh họa : Đổi tên ngành lưu trữ thành Index.html

Tiếp theo hãy chuột phải vào Index.Html và chọn “Edit with Notepad++”
